monty-carlo by mehmetalianil

Monte Carlo Simulation Module for Python. This module is meant to be a MC Simulation engine, in which users will be able to create their own systems and investigate parameters of their systems.

Dependencies

Depends on: Numpy Matplotlib mpl_toolkits Is not tested on Windows OS yet.
